# Excel_Sales_Dashboard

Project Overview:
This project is an interactive Sales Analytics Dashboard built using Microsoft Excel.
The dashboard analyzes transactional sales data and presents key business insights through visualizations and dynamic filters.
The objective of this project is to transform raw sales data into a clear and interactive dashboard that helps understand sales performance, customer behavior, and operational efficiency.

Dataset:
The dataset contains transactional information including:
Order ID
Product category
Quantity ordered
Revenue amount
Order date and shipment date
Customer gender
Purchase channel (App / Website / Partner platform)
Customer satisfaction rating
Delivery location

Key KPIs:
The dashboard tracks the following business metrics:
Total Orders
Total Quantity Sold
Total Revenue
Average Customer Rating
Average Delivery Time
Dashboard Insights

The dashboard provides insights such as:
Weekly sales trends
Product popularity based on quantity sold
Customer purchasing behavior by channel
Gender-based purchase analysis
Order quantity distribution per transaction
Delivery time analysis
Customer satisfaction trends
Geographic sales distribution

Tools & Features Used:
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers for interactive filtering
Conditional Formatting (Heatmap)
Map Charts
Calculated Columns

Key Features of the Dashboard:

Interactive filtering using slicers
Dynamic KPI summary section
Automated updates when data is refreshed
Visual analysis of sales performance and customer segments

Project Goal:
The goal of this dashboard is to demonstrate how Excel can be used as a powerful data analytics and business intelligence tool to convert raw transactional data into actionable insights.
